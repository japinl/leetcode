# 859. Buddy Strings

Given two strings `A` and `B` of lowercase letters, return `true` if and only if
we can swap two letters in `A` so that the result equals `B`.

__Example 1:__

```
Input: A = "ab", B = "ba"
Output: true
```

__Example 2:__

```
Input: A = "ab", B = "ab"
Output: false
```

__Example 3:__

```
Input: A = "aa", B = "aa"
Output: true
```

__Example 4:__

```
Input: A = "aaaaaaabc", B = "aaaaaaacb"
Output: true
```

__Example 5:__

```
Input: A = "", B = "aa"
Output: false
```
 

__Note:__

1. `0 <= A.length <= 20000`
2. `0 <= B.length <= 20000`
3. `A` and `B` consist only of lowercase letters.
